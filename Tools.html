Tools & Checklists

Things to check & remember

ABCD Check

Here are a few ABCD's to remember while designing a website. This is a mnemonic we created so that you never forget.

  1. A - Alt Text. Always remeber Alt text in html
  2. B -Be Structured. Always structure your html for easy screen reading
  3. C - Check colour. Check colour pallette, contrast and centered content
  4. D - Design Consistent. Be mindful of visual and congitive disabilities and design consistent
  5. E - Enlarge Content. Large text is always easy to read.
  6. F - Follow WCAG. Follow these guidelines.

SLIDE Check

SLIDE Check is another checklist that one can follow to check parameters for accesibility friendliness.

  1. S - Style. Heading styles and descriptive titles
  2. L -Link. Use embeded hyperlinks
  3. I - Images. Use Alt text to describe images
  4. D - Design. Clear simple and consistent design
  5. E - Evaluation. Use accesibility checkers
